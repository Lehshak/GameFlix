About This Project

This is a web application designed to make fun trivia games and also to help me learn more about node.js and mongodDB databases.

This project is built with bootstrap and node.js with express.

Web pages:

Here are some of the pages in this website :

  • Trivia Games: Test your knowledge of movies, actors, and trivia!
  • Movie Popularity Game: Compare movie ratings and see who wins!
  • Game Discovery: database for steam games.
  • Movie Listings: database for imdb movies.
